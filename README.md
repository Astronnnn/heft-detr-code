# HEFT-DETR-MAIN
This repository contains the HEFT-DETR project implementation.

## Project Structure

- `DatasetA&B/` - DatasetA & DatasetB # Liu, F., Wang, H., Feng, P., & Zeng, L. (2024). Integrated circuit packaging defect analysis and deep learning detection method. IEEE Transactions on Components, Packaging and Manufacturing Technology.
- `dataset_NEU-DET/` - NEU-DET dataset # He, Y., Song, K., Meng, Q., & Yan, Y. (2019). An end-to-end steel surface defect detection approach via fusing multiple hierarchical features. IEEE transactions on instrumentation and measurement, 69(4), 1493-1504.
- `dataset_visdrone/` - VisDrone dataset # Du, D., Zhu, P., Wen, L., Bian, X., Lin, H., Hu, Q., & Zhang, L. (2019). VisDrone-DET2019: The vision meets drone object detection in image challenge results. In Proceedings of the IEEE/CVF international conference on computer vision workshops.
- `ultralytics/` - Ultralytics models and utilities
- `train.py` - Training script
- `val.py` - Validation script
- `setup.py` - Setup configuration
